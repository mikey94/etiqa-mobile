<h1>Etiqa Test Mobile version</h1>

## Get started

1. Clone the repository

   ```bash
   git clone <repository-name>
   ```
   
2. Install dependencies

   ```bash
   npm install
   ```

3. Start the app

   ```bash
   npx expo start
   ```
